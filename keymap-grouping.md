# Allocation of modifier combinations

## 0 key (1)
`null`

## 1 key (4) - for __all apps__
`Meta`
`Shift`
`Ctrl`
`Cmd`

## 2 keys (6)
`Ctrl + Meta`
- __Rectangle__
    - half `hjkl`
    - quarters `ui89`

`Ctrl + Shift`
- __all apps__ (mainly vim)

`Meta + Shift`
- __all apps__ (mainly vim)

`Cmd + Ctrl`
- MacOS: App shortcuts
- __iterm__: 
    - `Arrows` resize
- __Aerospace__:
    - `hjkl` navigate

`Cmd + Meta`
- __iterm__:
    - `Arrows` for pane change

`Cmd + Shift`
- MacOS: 3-5 for screenshots
- __iterm__:
    - `Arrows` and `[]` for tab change
    - `D` for pane split
    - `Enter` for maximize

## 3 keys (4)
`Cmd + Meta + Shift`

`Cmd + Ctrl + Meta`

`Cmd + Ctrl + Shift`
- __Aerospace__:
    - `hjkl yuio` move windows
    - `nm,.` swap windows

`Ctrl + Meta + Shift` (MEH)
- __Rectangle__:
    - move to edge `hjkl`
    - sixths `uio890`

## 4 keys (1)
`Cmd + Ctrl + Meta + Shift` (HYPER)
- __Rectangle__:
    - fourths `hjkl`
